Real-Time Architecture

📡 Dual Real-Time Layers

Two complementary real-time systems run in parallel for maximum reliability:

Socket.io
Low-latency server push for GPS positions and status changes (<100ms). GPS updates every 10 seconds without touching DB.
Supabase Realtime
PostgreSQL CDC for persistent data changes (incidents, beds) as a reliable fallback. Row-level security enforced.
